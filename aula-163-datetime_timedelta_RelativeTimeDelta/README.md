📘 Módulo: datetime.timedelta e dateutil.relativedelta
Este diretório contém exemplos práticos sobre cálculo e manipulação de datas em Python utilizando:

datetime.timedelta (módulo nativo do Python)

dateutil.relativedelta (biblioteca externa mais flexível)

Os arquivos demonstram como somar, subtrair e comparar datas, além de converter strings em objetos datetime.

📂 Arquivos incluídos
aula_163_a_datetime.timedelta.py  
Mostra como usar timedelta para adicionar ou subtrair dias, horas, minutos e segundos.
Inclui exemplos de cálculos diretos entre datas.

aula_163_b_dateutil_relativedelta.py  
Demonstra o uso de relativedelta, que permite operações mais avançadas como adicionar meses, anos ou combinações complexas.
Também mostra como calcular diferenças entre datas com maior precisão.

🧠 Conceitos abordados
Conversão de strings para datetime com strptime

Diferença entre timedelta e relativedelta

Cálculo de intervalos entre datas

Adição e subtração de períodos

Manipulação de anos, meses, dias, horas e segundos

Formatação de datas com strftime
