📘 Módulo Datetime — Python (Aula 162) Este diretório contém exemplos práticos da seção de manipulação de datas e horas em Python, utilizando o módulo nativo datetime.

📂 Arquivos incluídos aula_162_a_criando_data_hora_moduto_datetime.py
Demonstra como criar objetos datetime, definir datas manualmente e trabalhar com ano, mês, dia, hora e minuto.

aula_162_b_DatetimeNow.py
Mostra como obter a data e hora atual usando datetime.now() e como formatar a saída.

aula_162_c_time_stamp.py
Explica o conceito de timestamp, como obtê-lo e como converter timestamps para objetos datetime.

🧠 Conceitos abordados Importação do módulo datetime

Criação de datas e horas manualmente

Uso de datetime.now()

Manipulação de atributos (ano, mês, dia, etc.)

Conversão entre datetime e timestamp

Formatação de datas com .strptime()
