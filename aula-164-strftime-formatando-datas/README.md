📘 Aula 164 — Formatando Datas com strftime
Este diretório contém o arquivo da aula 164 do curso de Python, dedicado ao uso do método strftime para formatar datas e horas em diferentes padrões.

O foco desta aula é aprender a transformar objetos datetime em strings formatadas, permitindo exibir datas no formato desejado.

📂 Arquivo incluído
aula_164_formatando_datas_strftime.py  
Demonstra:

Como converter uma string em datetime usando strptime

Como formatar datas com strftime

Como gerar diferentes padrões de exibição

Como acessar partes específicas da data (ano, mês, dia, hora, minuto, segundo)

🧠 Conceitos abordados
🔹 datetime.strptime()
Converte uma string para um objeto datetime seguindo um formato específico.

🔹 datetime.strftime()
Formata um objeto datetime para string no padrão desejado.

Exemplos comuns:

%d/%m/%Y → 13/12/2022

%Y-%m-%d → 2022-12-13

%H:%M:%S → 07:59:23
